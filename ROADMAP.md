# ShengJi Roadmap

ShengJi is developed around local processing, transparent behavior, reliable long-running tasks, and a native macOS experience. This roadmap describes priorities, not promised dates.

## Current priorities

- Developer ID signing and Apple notarization for a smoother public installation flow.
- Short, reproducible performance and accuracy comparisons for supported recognition engines.
- Continued reliability work for long recordings, recovery, model downloads, and exports.
- More complete live-caption workflows, including translation when the platform APIs and product quality allow it.
- Additional interface localizations contributed with matching documentation and testing.
- Better onboarding for choosing an engine on macOS 15.5–25.

## Areas open to feedback

- Recognition languages and model choices that matter most to real workflows.
- Subtitle editing and export improvements for creators, interviews, lectures, and accessibility use.
- CLI ergonomics and automation examples.
- Packaging and distribution options that preserve the project's privacy and licensing boundaries.

## Non-goals and constraints

- Intel Mac support is not currently planned.
- Apple SpeechAnalyzer recognition and floating live captions require macOS 26.
- The project will not claim a single “best” recognition engine without reproducible benchmarks.
- Recognition audio and imported transcripts should not be uploaded by the app.

Please use [GitHub Issues](https://github.com/maddylaneeee/ShengJi/issues) for concrete bugs and feature proposals.
