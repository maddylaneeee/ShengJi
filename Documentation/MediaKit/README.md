## Suggested sending order

1. Send the short message, GitHub link, and social card first.
2. If the recipient is interested, follow with the real screenshots, installation graphic, and project brief.
3. Disclose Apple silicon support, the macOS 26 feature boundary, and the current unnotarized build.
4. Invite an independent evaluation rather than asking the recipient to repeat prepared claims.
