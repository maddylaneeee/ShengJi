# ShengJi Project Brief

## One sentence

ShengJi is a native, open-source, local-first macOS app that turns microphones, media files, and the sound playing on a Mac into editable text and subtitles.

## Short description

ShengJi provides local transcription, floating Mac-audio captions, transcript editing, optional on-device translation, and subtitle export for Apple silicon Macs. The app does not upload recognition audio or imported transcripts.

## Highlights

- Floating local captions for Mac audio;
- microphone, audio, and video transcription;
- Apple Speech, Whisper, SenseVoice, and Parakeet engines;
- transcript search, replacement, range trimming, and multi-format export;
- Apple Translation and optional local NLLB;
- English and Simplified Chinese interfaces;
- MIT-licensed application source.

## Requirements and limits

- Apple silicon (`arm64`); no Intel support;
- macOS 15.5 minimum;
- Apple SpeechAnalyzer and floating live captions require macOS 26;
- on macOS 15.5–25, users must select a third-party model;
- SenseVoice and Parakeet currently support file transcription only;
- the public binary is not Developer ID signed or notarized.
